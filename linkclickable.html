HTML stands for HyperText Markup Language.
It is the standard language used to create web pages.
HTML provides the basic structure of a webpage using various elements.
These elements are called tags, and they tell the browser how to display content.
Each HTML tag is usually written in angle brackets like <tag>.
Most tags have an opening tag and a closing tag.
The closing tag includes a forward slash, like </tag>.
For example, a paragraph is written as <p>Your text</p>.
HTML documents start with a <!DOCTYPE html> declaration.
The main structure of an HTML page includes <html>, <head>, and <body> tags.
Inside the <head>, you can add metadata, links to stylesheets, and the page title.
The content that appears on the page goes inside the <body> tag.
Common tags include <h1> to <h6> for headings and <a> for links.
You can insert images using the <img> tag with a `src` attribute.
Lists can be created using <ul> for unordered and <ol> for ordered lists.
Each list item is wrapped in an <li> tag.
You can also create tables using <table>, <tr>, <td>, and <th> tags.
HTML forms are made with the <form> tag, which can include inputs, checkboxes, and buttons.
Comments in HTML are written like <!-- This is a comment -->.
Whitespace like extra spaces and line breaks are ignored in HTML.
To style HTML elements, you can use CSS either inline, in the head, or through an external file.
HTML5 introduced many new tags like <section>, <article>, and <nav> for better structure.
HTML is not a programming language—it's a markup language.
It focuses on describing the content and structure, not logic.
Understanding HTML is the first step toward learning web development.
It works hand-in-hand with CSS and JavaScript to build interactive websites.
